Mailing lists:


  • sal@csl.sri.com : Sal mailing list, it is intended for general discussions involving SAL, as well as announcements of new tools, language changes, etc. Subscribe by sending a message with 'subscribe' in the body to sal-request@csl.sri.com.
  • sal-devel@csl.sri.com : Sal developer mailing list, it is intended for discussions involving SAL development. Subscribe by sending a message with 'subscribe' in the body to sal-devel-request@csl.sri.com.
  • sal-bugs@csl.sri.com : Bug report list.
