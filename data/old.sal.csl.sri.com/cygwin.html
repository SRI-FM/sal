SAL 2.0 Cygwin Installation

Cygwin is a Unix-like environment that runs under Windows.
  • If you do not have it already, you first need to install the Java runtime
  • Then make sure it is in your Windows path: open a Command Prompt (MSDOS) window and type "java". If you get no response, go to Start -> Settings -> Control Panel -> System -> Advanced -> Environment Variables, highlight path and press edit and append a semicolon and the location of the java runtime (probably c:\Program Files\j2sdk_nb\jvm\bin). Start a new command prompt window and check for "java" again.
  • Next, install Cygwin from www.cygwin.com (version >= 1.5.3) and check that "java" works from there, too.
  • Then download the SAL Cygwin tar file into your Cygwin directory.
  • In Cygwin, type tar xzf sal-2.0-bin-i686-pc-cygwin.tar.gz, then cd sal-2.0 and ./install.sh.
  • Copy your license certificate to this directory, then type sal-smc and check that it seems to be working.
