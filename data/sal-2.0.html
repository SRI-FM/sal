Download SAL 2.0

Binary Distribution

By downloading the SAL binaries you agree to the terms of the SAL License, which forms part of the SAL Registration Form. On completing this form, a license certificate will automatically be generated and emailed to you. The binaries will not work without the certificate.

Note that the SAL distribution includes ICS; registration for SAL also covers ICS and the email includes certificates for both systems. Each certificate is for the single user named within it, but it is not restricted to any particular host or platform. If you already have a valid certificate, just download the distribution(s) you want.

The SAL binary distribution package contains: SAL2XML 2.0 (parser), SALenv 2.0 (model checkers and other tools), and ICS 2.0.

  • Linux (Intel)
  • Solaris (Sparc)
  • Mac OSX
  • Cygwin

Note: You must have Java runtime installed on your system, since SAL2XML needs a Java virtual machine.

This is all you need to run SAL (and ICS) on most systems. After you've completed the online registration, you'll be able to run the distribution right away.

Source Distribution

Before downloading any SAL source component, you must fill in the FM license and send it back to SRI. After processing your license, a password will be mailed to you, allowing you to access the sources. In the meantime, you may use the binaries immediately after a simple registration process.
