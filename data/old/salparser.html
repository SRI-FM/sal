A SAL Parser:


One of the SRI SAL Parsers is written in ANTLR and generates JAVA code that translates from the concrete syntax described in the language report into the abstract syntax described in the SAL DTD. This may then be read by any language that has support for DOM or SAX. Even if this support is missing, it is much easier to define a parser for the SAL abstract syntax than for the concrete syntax.
