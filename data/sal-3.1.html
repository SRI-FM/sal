SAL Version 3.1 (released March 7, 2012)

SAL 3.1 includes SALenv 3.0 (symbolic, bounded, and infinite-bounded model checkers, and related tools), and, optionally, Yices.

SAL 3.1 Sources

See the included README file for build instructions.

SAL 3.1 Binaries with Yices

This is probably the version you want, as some SAL features will not work without Yices. Yices requires agreement to a "click-through" license.
  • Linux 2.6/i686 (32bit Intel Pentium and compatible)
  • Linux 2.6/x86_64 (64bit AMD/Intel and compatible)
  • Mac OS X 10.6 (compatible with Leopard, Snow Leopard, and Lion)
  • Cygwin (Windows - 32bit Intel)

SAL 3.1 Binaries without Yices

  • Linux 2.6/i686 (32bit Intel Pentium and compatible)
  • Linux 2.6/x86_64 (64bit AMD/Intel and compatible)
  • Mac OS X 10.6 (compatible with Leopard, Snow Leopard, and Lion)
  • Cygwin (Windows - 32bit Intel)
